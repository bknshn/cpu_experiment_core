# コア
2017年CPU実験5班のコアです。

## ファイル説明
- src/ip/ : IPコアの設定
- src/computer.xdc : 制約ファイル(Constrains)
- src/top.sv : トップ
- src/core.sv : コア本体
- src/fib3 : fib3の命令バイナリが入った命令メモリ
